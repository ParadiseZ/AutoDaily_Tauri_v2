# 前端项目结构说明文档

- 生成范围：`src/` 下前端源码文件，共 213 个文件。
- 说明：保留全项目索引，同时给执行流相关文件追加“定位”标记；`主线`表示直接进入运行链，`支撑`表示直接服务主线，`外围`表示主要承担项目其他职责。

## 主线阅读顺序

1. `src/views/ScriptEditor.vue`
   定义层编辑与调试运行入口。
2. `src/views/TaskManagement.vue`
   正式运行入口、assignment 管理入口、模板值入口。
3. `src/views/task-management/TaskDevicePanel.vue`
   设备运行控制、调度记录、运行事件的汇合面板。
4. `src/services/runtimeService.ts`
   `cmd_sync_device_runtime_session`、`cmd_run_script_target` 的前端入口。
5. `src/services/taskService.ts`
   assignment、schedule、time template 的前端入口。
6. `src/services/scriptTemplateValueService.ts`
   `script_time_template_values` 的前端入口。
7. `src/store/runtime.ts`、`src/store/task.ts`、`src/store/device.ts`
   运行态、调度态、设备态在前端的状态收口。
8. `src/types/app/domain.ts`
   前端本地类型和运行时合同镜像的混装入口，优先看 `RunTarget`、`Runtime*Event`、`ScriptTimeTemplateValuesDto`。
