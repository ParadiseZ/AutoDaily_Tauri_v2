# 视觉测试第二版设计

## 目标

在第一版“目录图像 + 设备截图 + 独立窗口 + 检测/OCR + 取色/区域采样 + 基础过滤 + 本地保存”的闭环之上，补齐更偏工作流联动的能力，但不引入截图历史。

## 第二版范围

1. 区域采样增强
   提供多区域对比、固定采样模板、区域命名和复用。

2. 颜色误差比对增强
   支持多目标颜色、阈值预设、命中/未命中状态标记，以及按色差排序。

3. 结果导出
   支持导出 OCR/Det 结果、颜色采样结果、当前模型参数和选中图像元数据。

4. 结果反向定位
   从右侧 OCR / Det 结果点击后自动定位到中间图像对应区域，并做高亮聚焦。

5. 规则回填
   从 OCR 结果、检测标签、颜色分析结果一键生成脚本规则草稿，回填到脚本编辑器的视觉条件配置。

6. 筛选增强
   增加标签筛选、分数区间筛选、文本正则筛选和多条件组合过滤。

7. 模型预设管理
   支持把当前检测/OCR参数保存成视觉测试预设，并可按脚本或通用模板切换。
