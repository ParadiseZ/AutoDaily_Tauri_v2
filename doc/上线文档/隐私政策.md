## 2. 保存、共享与安全

本地数据保存于应用数据目录；账户会话、资料缓存、脚本配置和 SMTP 配置会存于本机。账户和云端脚本数据保存于运营者的服务器，期限为实现目的所需的最短时间或法律要求的期限，具体期限：[待填写并与后端备份策略一致]。

我们不会出售个人信息。仅在提供服务所必需、取得有效同意、履行法定义务或处理争议时，向受托处理者或有关机关提供必要信息；现有已知对象见《第三方个人信息共享清单》。我们采取传输加密、访问控制、最小权限等合理措施，但请不要将密码、支付凭证等高风险信息写入脚本、日志或截图。

## 3. 您的权利

您可通过客户端修改账户信息、退出登录、删除本地脚本和配置；也可联系运营者请求查阅、复制、更正、删除个人信息、撤回同意、注销账户或解释处理规则。我们将在法定期限内处理。注销后，我们将删除或匿名化相关个人信息，法律另有规定或争议处理需要保留的除外。

撤回非必要同意不影响此前基于同意开展处理的效力，但可能使对应功能无法使用。账号注销与本机文件删除是不同操作；请先备份您需要的本地脚本。

## 4. 未成年人、更新与投诉

本服务不向未满 14 周岁者提供账户、云端脚本或赞助服务。若监护人发现我们在未取得有效同意的情况下处理了儿童个人信息，请按上述联系方式通知，我们将核验后及时处理。

本政策更新时会以客户端公告或其他合理方式提示。对重要变更，我们将依法重新取得同意。您也可向网信等有权机关投诉举报。
