# 项目结构说明文档

- 生成范围：`src-tauri/build.rs`、`src-tauri/src/**/*.rs`、`src-tauri/crates/**/src/**/*.rs`，共 236 个 Rust 文件。
- 说明：保留全项目索引，同时给执行流相关文件追加“定位”标记；`主线`表示直接进入运行链，`支撑`表示直接服务主线，`外围`表示主要承担项目其他职责。

## 主线阅读顺序

1. `src-tauri/crates/runtime_common/src/ipc/message.rs`
   先看 `RunTarget`、`RuntimeSessionSnapshot`、`RuntimeExecutionPolicy`、`Runtime*Event` 这些运行时合同。
2. `src-tauri/src/api/domain/scripts.rs`、`src-tauri/src/api/domain/schedule.rs`
   这是定义层、调度层和模板覆盖值进入主进程的入口。
3. `src-tauri/src/api/infrastructure/runtime_sync.rs`
   在线设备 session reload 的主收口点。
4. `src-tauri/crates/child_support/src/infrastructure/session/runtime_session.rs`
   child 对 session baseline 的本地容器。
5. `src-tauri/crates/child_support/src/infrastructure/context/runtime_context.rs`
   执行态、观察态和服务依赖当前的组织方式。
6. `src-tauri/crates/child_support/src/infrastructure/scripts/execution_plan.rs`
   正式队列和调试运行进入 child 后的计划装配入口。
7. `src-tauri/crates/child_support/src/infrastructure/scripts/scheduler.rs`
   child 调度主循环。
8. `src-tauri/crates/child_support/src/infrastructure/scripts/executor.rs` 及 `executor/*`
   真正的步骤执行、策略执行、timeout、作用域和变量处理主链。
9. `src-tauri/crates/child_support/src/infrastructure/ipc/runtime_reporter.rs`
   runtime event 上报链入口。
